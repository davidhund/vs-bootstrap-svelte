# Getting Help

## I have a question about the template

→ Use [GitHub Discussions](https://github.com/davidhund/vs-bootstrap-svelte/discussions)

Ask about:
- How to use a feature
- Best practices for Svelte/SvelteKit
- Recommendations for extending the template
- General setup questions

## I found a bug

→ Open a [GitHub Issue](https://github.com/davidhund/vs-bootstrap-svelte/issues)

Include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your environment (OS, Node.js version, etc.)

## I want to report a security vulnerability

→ See [SECURITY.md](./SECURITY.md)

**Do not** open a public issue.

## I want to contribute an improvement

→ See [CONTRIBUTING.md](./CONTRIBUTING.md)

## I'm using this template for a project and need help

This template comes as-is. For projects *created from* this template, you'll need to:
- Check the [SvelteKit docs](https://kit.svelte.dev)
- Ask in the [Svelte Discord](https://discord.gg/svelte)
- Open issues in your own project's repository

## Something else?

Feel free to start a [discussion](https://github.com/davidhund/vs-bootstrap-svelte/discussions) and we can figure it out.
